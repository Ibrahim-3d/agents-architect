# Code of Conduct

## Short version

Be decent. Focus on the work. Assume good faith. Call out bad faith.

## Longer version

This project adopts the spirit of the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/). Harassment, personal attacks, and discrimination are not tolerated. Technical disagreement is welcome and expected — attack the idea, not the person.

## Enforcement

Report issues to: papsiman5@gmail.com

Maintainers may warn, remove content, or ban contributors whose behavior makes the project unwelcoming. Decisions are final but may be revisited.
